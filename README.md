# E-commerce Sales Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on E-commerce sales data to uncover trends, patterns, and business insights.

## Objectives
- Analyze product sales
- Study payment methods
- Identify revenue trends
- Understand customer behavior

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed

### Product Analysis
- Product Sales Distribution
- Revenue by Product

### Payment Analysis
- Payment Method Distribution

### Order Analysis
- Order Status Distribution

### Customer Analysis
- Top Customers by Spending

### Time Analysis
- Monthly Revenue Trend

### Marketing Analysis
- Referral Source Performance
- Coupon Usage Analysis

## Visualizations
- Bar Charts
- Pie Charts
- Line Charts
- Box Plots

## Key Insights
- Identified top-selling products
- Found highest revenue generators
- Analyzed customer spending patterns
- Tracked monthly revenue growth
